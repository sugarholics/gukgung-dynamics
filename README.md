# 국궁 3D 동역학 시뮬레이션

한국 전통 활(국궁) 개량궁의 물리 기반 3D 모델 및 탄성 동역학 시뮬레이션.

React + Three.js(r128)로 구현되었으며, 활의 구조·재료 파라미터로부터 brace height, 당김력 곡선, 발시 후 진동까지 물리적으로 계산합니다.

## 구조 모델

```
양양고자 ─── 고자(강체) ─── 도고자 ┐
                                   활채(탄성보)
                  줌통(강체, V자) ──┤
                                   활채(탄성보)
양양고자 ─── 고자(강체) ─── 도고자 ┘
```

- **줌통(grip/대림)**: 중앙 강체, V자 형태 (`gripAngle` 파라미터로 리커브 프로파일 조절)
- **활채(limb)**: Euler-Bernoulli 탄성 보, 호 길이 보존 곡률 적분 모델. 두께·폭이 줌통에서 끝으로 갈수록 감소 (taper)
- **고자(siyah)**: 활채 끝에 고정 각도로 연결된 강체

## 시위 모델 — 도르래 감김

시위 경로는 도고자(활채/고자 접합부)를 **밧줄이 기둥에 걸리듯** 취급하여 동적으로 판정합니다.

- **감김 모드**: 양양고자 → 고자면 접촉 → 도고자 → 자유구간 → nock → 도고자 → 양양고자
- **직진 모드**: 양양고자 → nock → 양양고자 (도고자 접촉 없음)

매 상태마다 양양고자→nock 직선 경로 위 도고자의 위치를 확인하여, 도고자가 직선 앞에 돌출하면 감김, 뒤에 있으면 직진. 당기는 중 시위와 고자의 상대 기하학 변화에 따라 자연스럽게 전환됩니다.

## 물리 엔진

### 정적 해석
- **곡률 적분**: 자연 곡률(반곡) + 하중 곡률(캔틸레버 모멘트)를 호 길이 보존하며 적분
- **Brace solver**: `computeNockX`의 null→valid 전환점을 이분법으로 탐색하여 현걸이 평형 결정
- **당김력**: `F = 2 × loadFactor × (nockX − pivot.x) / |pivot.y|` (pivot = 감김 모드에 따라 도고자 또는 양양고자)

### 동적 해석 (발시 후 진동)
- **k_eff**: brace 근방 당김력 구배
- **m_eff**: 1차 굽힘 모드 유효질량 `2 × (0.236 × m_limb + m_siyah)`
- **고유진동수**: `ω₀ = √(k_eff / m_eff)`, 감쇠비 적용
- **진폭**: 화살 반동 에너지 → 줌통 진동 진폭으로 환산

## 개량궁 기본 파라미터

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| bowLength | 1.20 m | 활 전체 길이 |
| gripLength | 0.12 m | 줌통 길이 |
| siyahLength | 0.08 m | 고자 길이 |
| siyahAngle | 55° | 고자 각도 |
| gripAngle | 10° | 줌통 V자 각도 |
| elasticModulus | 45 GPa | FRP/카본 탄성계수 |
| stringLength | 1.08 m | 시위 길이 |
| maxDraw | 0.80 m | 만작 거리 |
| reflexAngle | 35° | 반곡 각도 |

## 실행

React 환경에서 JSX 컴포넌트로 import하여 사용합니다. Three.js r128이 필요합니다.

```jsx
import KoreanBow3D from './국궁_3d_모델';

function App() {
  return <KoreanBow3D />;
}
```
